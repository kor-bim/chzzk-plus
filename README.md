## 소개

CHZZK Plus는 macOS Safari에서 치지직 라이브 시청 환경을 더 편리하게 만드는
Safari 웹 확장 프로그램입니다. 치지직의 기본 디자인과 조화를 유지하면서 필요한
기능만 가볍게 추가합니다.

## 주요 기능

- 선호 화질 자동 고정 및 플레이어 재렌더링 시 자동 재적용
- 강도를 조절할 수 있는 화면 선명도 필터
- 라이브 구간을 확인하고 이동할 수 있는 재생바
- 현재 화질, 레이턴시 및 플레이어 상태 표시
- 스트림 통계와 우클릭 통계 보기
- 네트워크 응답 처리, 자동 건너뛰기 및 감지 화면 제거를 포함한 광고 차단
- 순위권 채팅 숨김
- 블라인드 채팅 복구 및 원문 표시

## 디자인

치지직의 Neon Green `#00FFA3`과 Black `#000000`을 기반으로 하며, 플레이어
내부 UI와 확장 프로그램 팝업이 기존 서비스에 자연스럽게 어울리도록 구성했습니다.

## 지원 환경

- macOS
- Safari
- `chzzk.naver.com`

iOS는 지원하지 않습니다.

## 참고

CHZZK Plus는 치지직 또는 NAVER와 관련 없는 비공식 개인 프로젝트입니다. 치지직의
페이지 구조나 API가 변경되면 일부 기능이 일시적으로 작동하지 않을 수 있습니다.
